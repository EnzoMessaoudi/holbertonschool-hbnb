<h1 align="center"> HBNB projet for Holberton School </h1>

<p align="center">
In this Project, we will create an application similar to the Airbnb site. In this application, the user can see multiple places that he can rental and review. He can also create an account then post on the site the places he can rent and the aminity that are selled with it.
</p>

## Technologies used
- Python  
- HTML/CSS  
- JavaScript
- RESTFUL API
- Bcrypt
- JWT

## How to install it ?
You can see and install everything in the requirements.txt file in any part of the project.

## How to launch it ?
You can launch all the application with a simple command: python3 run.py  
The main page is index.html. You can go on any navigator to see the application.
